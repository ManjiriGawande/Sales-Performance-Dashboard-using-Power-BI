# Sales-Performance-Dashboard-using-Power-BI
“I created this Sales Performance Dashboard in Microsoft Power BI to analyze sales performance, profit, customer behavior, and sales forecasting using retail store data.
In the first dashboard, I used KPIs like Total Sales (1.6M), Orders (22K), Profit (175K), and Average Ship Days (4) to track overall business performance. I also created visualizations for:
Sales by Category and Sub-Category
Sales by Region and State
Payment Mode and Ship Mode Analysis
Monthly Sales and Profit Trends
I added slicers for regions like Central, East, South, and West to make the dashboard interactive and help users filter insights easily.
In the second dashboard, I built a 15-day Sales Forecast using Power BI forecasting features to predict future sales trends based on historical data. This helps businesses in planning inventory and sales strategies.
For this project, I used:
Data Cleaning and Transformation in Power Query
Data Modeling
DAX Measures for KPIs
Interactive Visualizations and Forecasting
This project improved my skills in data visualization, business analysis, and dashboard design.”
